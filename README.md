Mi Proyecto: Liga de Esports

He creado una página web funcional usando Python (Flask) y una base de datos MySQL en phpMyAdmin.

Esto es lo principal que hace:

- Login y Registro: Tengo una zona pública y una privada. Para ver los torneos, el usuario tiene que crearse una cuenta e iniciar sesión.

- Gestor de Torneos: En la zona privada, puedes crear nuevos torneos. Todo se guarda de verdad en la base de datos.

- Portafolio ("About Me"): He añadido una sección donde muestro los otros proyectos que hemos hecho en clase.

- Código limpio: He separado la base de datos en otro archivo (db_module.py) y he usado una plantilla base (base.html) para el menú, así el código es mucho más profesional.
